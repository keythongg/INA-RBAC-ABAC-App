# Preduslovi

## Osnovni Preduslovi za Pokretanje Aplikacije

Za pokretanje aplikacije **INA Digitalni Menadžment Sistem**, potrebno je osigurati sljedeće komponente:

### Instaliran Node.js i NPM
- **Node.js**: `22.13.1` (preporučena verzija)
- **NPM** (Node Package Manager): `11.1.0`

### Ključne Biblioteke i Alati
- **React**: `19.0.0`
- **React DOM**: `19.0.0`
- **Vite**: `6.1.0` 
- **React Router DOM**: `7.1.5` (za navigaciju)
- **Axios**: `1.7.9` (za API pozive)
- **Socket.io-client**: `4.8.1` (za real-time komunikaciju)

### Dodatni Sistemski Zahtjevi
- **Operativni sistem**: Windows 10/11, Linux, ili macOS
- **Pretraživač**: Google Chrome (preporučen) ili Firefox

---

Ovo su osnovni preduslovi bez kojih aplikacija ne može biti pokrenuta.

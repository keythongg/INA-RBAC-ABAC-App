# Instalacija

### Instalacija Zavisnosti
```bash
npm install
```
## Kompletna Instalacija Zavisnosti
Kopirajte i zalijepite sljedeće komande u terminal:

```bash
# React i development okruženje
npm install react react-dom vite

# State management i UI komponente
npm install @reduxjs/toolkit react-redux redux @mui/material @emotion/react @emotion/styled

# Routing
npm install react-router-dom

# API komunikacija
npm install axios swr

# Autentifikacija i sigurnost
npm install jsonwebtoken bcryptjs

# Stilizacija
npm install @emotion/styled @emotion/react

# Grafikoni i vizualizacija
npm install apexcharts react-apexcharts

# Real-time komunikacija
npm install socket.io-client

# Backend zavisnosti
npm install express sqlite3 cors dotenv

# Opcionalni alati
npm install docusaurus
```
:::warning Upozorenje
Osigurajte da imate Node.js (v22.13.1) i NPM (v11.1.0) instalirane.
:::

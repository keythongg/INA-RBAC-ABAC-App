# Uvod

INA Rafinerija - Dokumentacija za Digitalni Menadžment Sistem

<img src="/img/inadashboard.png" class="doc-image" alt="Dashboard" />

## O INA

Dashboard namijenjen isključivo internom korištenju od strane 
zaposlenika. Sistem omogućava centralizovan prikaz podataka, olakšava praćenje relevantnih informacija i 
optimizuje operativne procese unutar rafinerije.

Intuitivan korisnički interfejs osigurava jednostavan pristup ključnim podacima. Dashboard je dizajniran u skladu 
s najvišim industrijskim standardima za jednostavnost korištenja, brzinu i skalabilnost, koristeći modernu 
tehnološku infrastrukturu zasnovanu na **React Hooks, Components, Vite, Material UI v6, Express i Node.js**.
